== Description ==

Erm&ouml;glicht den Einbau des Affiliando Vergleichsrechner mittels [Shortcode]:

* Einfache Handhabung ohne Programmierkenntnisse
* Cached alle CSS- /Javascript-Dateien und Bilder lokal

* Alle Einstellungsm&ouml;glichkeiten die der PHP-Include erm&ouml;glicht sind mittels Shortcode machbar.


== Installation ==

1. Hochladen des Ordners in den WordPress Ordner `/wp-content/plugins/`
2. Aktivieren des 'Plugins' &uuml;ber das Pluginmenu in WordPress
3. Benutze das Plugin mittesl Shortcode und Lorem Ipsum Admin-Menu
